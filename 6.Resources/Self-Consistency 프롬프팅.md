# Self-Consistency 프롬프팅

## 🎯 기법 개요

**정의**: 동일한 문제에 대해 여러 개의 독립적인 추론 경로를 생성하고, 결과들의 일관성을 검증하여 가장 신뢰할 수 있는 답변을 도출하는 기법

**핵심 원리**:
- 다중 추론 경로를 통한 답변의 안정성 확보
- 투표 메커니즘을 통한 최종 결과 선택
- 추론 과정의 다양성과 결과의 일관성 균형

**주요 특징**:
- 단일 추론의 편향성과 오류 위험 감소
- 복잡한 문제에 대한 신뢰도 향상
- 다각도 관점을 통한 포괄적 분석
- Chain-of-Thought와 결합하여 강력한 추론 능력 제공

---

## 🔧 기법 구조

### 기본 프로세스
1. **다중 추론 경로 생성**: 동일 문제에 대해 3-5개의 서로 다른 접근 방식 적용
2. **독립적 추론 수행**: 각 경로별로 독립적인 논리적 추론 진행
3. **결과 집계**: 각 경로에서 도출된 결과를 투표 방식으로 집계
4. **일관성 검증**: 가장 많은 지지를 받은 답변을 최종 결과로 선택

### Self-Consistency 실행 원칙
1. **독립성**: 각 추론 경로는 서로 다른 관점에서 독립적으로 수행
2. **일관성**: 여러 경로에서 동일하게 선택된 결과의 신뢰도가 높음
3. **투명성**: 모든 추론 과정과 투표 결과를 명시적으로 공개
4. **근거 중심**: 각 선택의 구체적이고 논리적인 근거 제시

## 💡 시스템 프롬프트 예시

### 마케팅 채널 선정 Self-Consistency 시스템

```markdown
# 마케팅 채널 전략 전문가 Self-Consistency 프롬프트

## 역할
당신은 **마케팅 채널 전략 전문가**입니다. Self-Consistency 방법론을 사용하여 다중 추론 경로를 통해 신뢰할 수 있는 마케팅 채널 선정을 수행합니다.

## 작업 목표
**신규 비건 화장품 브랜드**의 런칭을 위한 최적의 마케팅 채널 3개를 선정하고, 각 채널의 선정 근거와 예상 효과를 도출하세요.

## Self-Consistency 실행 프로세스

### 다중 추론 경로 생성
동일한 문제에 대해 **3가지 서로 다른 접근 방식**으로 독립적인 추론을 수행한 후, 결과들의 일관성을 검증하여 최종 결론을 도출합니다.

### Few-shot 학습 가이드
다음 예시를 참고하여 각 접근 방식의 추론 패턴을 학습하세요:

예시 문제: 새로운 피트니스 앱의 마케팅 채널 선정

접근 방식 1 (고객 중심):
20-30대 직장인의 앱 사용 패턴 → 인스타그램, 유튜브, 네이버 블로그 선정

접근 방식 2 (제품 중심):
운동 콘텐츠 특성 → 유튜브, 틱톡, 인플루언서 마케팅 선정

접근 방식 3 (경쟁사 중심):
경쟁 앱들의 성공 채널 → 유튜브, 인스타그램, 구글 광고 선정

투표 결과: 유튜브(3표), 인스타그램(2표) → 핵심 채널로 선정

## 3가지 독립 추론 경로

### 추론 경로 1: 타겟 고객 행동 중심 접근
**분석 관점**: 20대 후반~30대 초반 여성의 미디어 소비 패턴과 구매 의사결정 과정을 중심으로 분석

**추론 단계**:
1. 타겟층의 디지털 미디어 이용 시간과 플랫폼 선호도 분석
2. 화장품 구매 시 정보 탐색 경로와 의사결정 요인 파악
3. 각 채널별 타겟층 도달률과 참여도 평가
4. 고객 여정(Customer Journey) 각 단계별 최적 채널 매핑

**예상 결론**: [3개 채널과 선정 근거]

### 추론 경로 2: 제품 특성 최적화 접근
**분석 관점**: 비건, 친환경, 피부 진정 효과라는 제품 특성을 가장 효과적으로 어필할 수 있는 채널 중심

**추론 단계**:
1. 비건/친환경 화장품의 핵심 메시지와 콘텐츠 형태 정의
2. 각 채널별 콘텐츠 형식과 제품 특성 부합도 평가
3. 제품 데모, 성분 설명, 브랜드 스토리 전달에 적합한 매체 분석
4. 신뢰성과 전문성 구축에 유리한 채널 우선순위 설정

**예상 결론**: [3개 채널과 선정 근거]

### 추론 경로 3: 경쟁사 벤치마킹 접근
**분석 관점**: 성공한 경쟁 비건 브랜드들의 채널 전략 분석과 시장 공백 기회 탐색

**추론 단계**:
1. 주요 비건 화장품 브랜드들의 마케팅 채널 포트폴리오 조사
2. 각 브랜드별 채널 성과와 투자 비중 분석
3. 경쟁이 치열한 채널과 상대적으로 공백인 채널 식별
4. 차별화 가능한 채널 조합과 포지셔닝 전략 수립

**예상 결론**: [3개 채널과 선정 근거]
```

## 🚀 활용 사례

### 적합한 작업 유형
- **중요한 의사결정**: 비즈니스 전략, 투자 결정, 리스크 평가
- **복합 분석**: 다요인 고려가 필요한 복잡한 문제
- **신뢰성 중요 업무**: 의료, 법률, 안전 관련 판단
- **전략적 선택**: 제품 개발, 시장 진입, 파트너십

### 부적합한 작업 유형
- **단순 질의응답**: 명확한 답이 있는 사실 확인
- **창작 작업**: 주관적 창의성이 중요한 업무
- **즉석 응답**: 빠른 응답이 필요한 상황

## ⚡ 장점 및 한계

### 장점
- **높은 신뢰성**: 다중 검증을 통한 오류 감소
- **편향 완화**: 단일 관점의 편향성 보완
- **투명한 과정**: 추론 과정과 근거가 명확히 드러남
- **안정적 결과**: 일관성 있는 고품질 답변 제공

### 한계
- **시간 소요**: 여러 추론 경로로 인한 처리 시간 증가
- **비용 증가**: 더 많은 토큰 사용으로 인한 비용 상승
- **복잡성**: 단순한 문제에는 과도한 과정
- **합의 어려움**: 모든 경로에서 다른 결과가 나올 경우 판단 곤란

## 🎨 변형 기법

### Weighted Self-Consistency
- 각 추론 경로에 가중치를 부여하여 투표
- 전문성이나 신뢰도에 따른 차등 반영

### Hierarchical Self-Consistency
- 상위 수준과 하위 수준의 계층적 일관성 검증
- 전체적 일관성과 세부적 일관성 동시 확보

### Dynamic Self-Consistency
- 추론 과정에서 실시간으로 일관성 모니터링
- 불일치 발견 시 즉시 추가 경로 생성

---

## 🔍 투표 메커니즘 설계

### 단순 다수결
- 가장 많은 지지를 받은 답변 선택
- 명확하고 직관적인 방법

### 가중 투표
```
채널 점수 = Σ(경로별 점수 × 경로 가중치)
경로 가중치 = 추론 품질 점수 × 전문성 점수
```

### 신뢰도 기반 선택
```
신뢰도 = (일치하는 경로 수 / 전체 경로 수) × 100%
- 고신뢰도 (80% 이상): 즉시 채택
- 중신뢰도 (50-79%): 추가 검토 후 채택
- 저신뢰도 (50% 미만): 추가 경로 생성 필요
```

## 📚 추가 학습 자료

### 실습 예제
1. **제품 론칭 전략**: 다각도 시장 분석을 통한 전략 수립
